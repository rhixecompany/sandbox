# profile — Blog/Profile Website

RhixeCompany Profile — a Django-based blog and profile website with rich content editing, cloud storage, and tagging.

## Tech Stack
- **Backend**: Python 3.10, Django 3.x, DRF
- **Content**: django-ckeditor, django-crispy-forms
- **Storage**: Google Cloud Storage / AWS S3
- **Database**: PostgreSQL

## Quick Start
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```
