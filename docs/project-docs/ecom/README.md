# ecom — Ecommerce Platform

RhixeCompany Ecommerce — a full-stack ecommerce platform with product catalog, shopping cart, reviews, PayPal payments, and admin dashboard.

## Tech Stack
- **Backend**: Python 3.10, Django 3.1, DRF, SimpleJWT
- **Frontend**: React 18, Redux, React Bootstrap
- **Payments**: PayPal
- **Storage**: AWS S3 / Google Cloud

## Quick Start
```bash
# Backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

# Frontend
cd frontend && npm install && npm start
```
