# Django-Scrapy-Selenium — Web Scraping Platform

A full-stack web scraping platform with Django, Scrapy, Selenium, and Celery.

## Tech Stack
- **Backend**: Python 3.10+, Django 4.x, DRF
- **Scraping**: Scrapy, Selenium WebDriver, BeautifulSoup4
- **Task Queue**: Celery + Redis/RabbitMQ
- **Frontend**: Webpack, Tailwind CSS, Alpine.js

## Quick Start
```bash
pip install -r requirements.txt
npm install
python manage.py migrate
python manage.py runserver
```
