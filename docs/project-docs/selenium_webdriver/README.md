# selenium_webdriver — Selenium Web Scraper

A Node.js Selenium WebDriver project for automating Chrome browser to extract comic/manga data.

## Tech Stack
- **Runtime**: Node.js 18+, ES Modules
- **Automation**: selenium-webdriver 4.x, ChromeDriver
- **Formatting**: Prettier

## Quick Start
```bash
npm install
npm test  # runs src/scrape.js
```

## Scripts
- `src/scrape.js` - Main scraper
- `src/scrape2.js` - Secondary scraper
- `src/test.js` - Test utility
