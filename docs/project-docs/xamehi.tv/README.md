# xamehi.tv — Movie/TV Streaming Platform

A movie and television streaming website with Django REST backend and React 17 frontend.

## Tech Stack
- **Backend**: Python, Django, DRF, SimpleJWT
- **Frontend**: React 17, Redux, Material-UI 4, react-admin
- **Payments**: PayPal
- **Media**: video-react player

## Quick Start
```bash
# Backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

# Frontend
cd frontend && npm install && npm start
```
