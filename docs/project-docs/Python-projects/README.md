# Python-projects — Beginner Python Scripts

A collection of 18 beginner-friendly Python scripts for learning and automation.

## Scripts Include
- Calculators, games, face detection, QR code generator
- Web scraper, URL shortener, email sender
- Dictionary tool, schedule automation, and more

## Quick Start
```bash
pip install -r requirements.txt
python basic_calculator.py
```
