> Extracted from `shuffle-json-data.prompt.md`.

## Acceptable JSON

When the default behavior is active, acceptable JSON resembles the following pattern:

```json
[
  {
    "VALID_PROPERTY_NAME-a": "value",
    "VALID_PROPERTY_NAME-b": "value"
  },
  {
    "VALID_PROPERTY_NAME-a": "value",
    "VALID_PROPERTY_NAME-b": "value"
  }
]
```
