# Core Skills Table

This skills table is shared across all fix/sync prompts. Each prompt
customizes the **Purpose** column for its domain.

| Skill                         | Purpose (fill per-prompt context)                              |
| ----------------------------- | --------------------------------------------------------------- |
| `context-map` (prompt)        | Map impacted files and dependencies before edits                |
| `brainstorming`               | Explore approaches before implementation                        |
| `plans-and-specs`             | Create structured plans with specs                              |
| `dispatching-parallel-agents` | Process work in parallel batches                                |
| `subagent-driven-development` | Delegate per-task execution                                     |
| `systematic-debugging`        | Identify issues and edge cases                                  |
| `simplify`                    | Remove duplicates and consolidate                                |
| `acpx-executor`               | Execute a prompt via any ACPX provider                          |
