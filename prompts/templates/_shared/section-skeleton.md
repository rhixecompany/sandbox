# Standard Prompt Section Skeleton

Use this skeleton when creating a new fix/sync prompt. Each section is
optional — omit sections that don't apply.

---

## Goal

A single sentence: what does this prompt accomplish?

## Context

When should this prompt be used? What problem does it solve?

## Inputs

What data or artifacts does the prompt expect?

- Thing 1
- Thing 2

## Outputs

What does the prompt produce?

- Thing 1
- Thing 2

## Rules

See `_shared/rules-core.md` for the core rule set. Add domain-specific
rules below.

1. ...
2. ...

## Skills Required

See `_shared/skills-table-core.md` for the shared skills table. List
only domain-specific skills here.

## Phases

### Phase 1: ...

Description of phase 1.

### Phase 2: ...

Description of phase 2.

## Steps

Detailed numbered steps within each phase.

## Tasks

Mapping of phases to tasks for delegation.

## Actions

Explicit action items with clear owners.
