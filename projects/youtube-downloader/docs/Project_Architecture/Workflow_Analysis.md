## Project Overview

**Project:** YouTube Downloader
**Type:** CLI Video/Audio Downloader
**Stack:** Python 3.x + yt-dlp + curl_cffi
**Architecture Pattern:** Standalone CLI Scripts
**Entry Point:** `main_noplaylist.py`, `main_playlist.py`, `main_loop_playlist.py`, `main_loop_noplaylist.py`

A Python CLI tool for downloading YouTube videos and playlists using yt-dlp and curl_cffi, with support for single videos, playlists, and batch loop mode.

## Workflows

### Workflow 1: Single Video Download

**Description:** User downloads a single YouTube video with format selection.

**Entry Points:**
- `python main_noplaylist.py`

**Steps:**
1. User runs `python main_noplaylist.py`
2. Script prompts for YouTube URL
3. Script fetches video metadata via yt-dlp
4. User selects download format (video quality, audio only, etc.)
5. Script downloads video using yt-dlp
6. Progress is displayed during download
7. Downloaded file is saved to local filesystem
8. Script may convert format via FFmpeg (if available)

**Error Handling:**
- Invalid URL → prompt again with validation
- Video unavailable/removed → clear error message
- Download interrupted → partial file cleanup, retry option
- FFmpeg missing → skip conversion, notify user
- Rate limiting → yt-dlp built-in delays and respect
- `.env` / URLs never committed to VCS

**Test Patterns:**
- `python test.py` — basic tests
- `ruff check . && mypy *.py` — linting and type checking

### Workflow 2: Playlist/ Batch Loop Download

**Description:** User downloads an entire YouTube playlist or multiple videos in batch/loop mode.

**Entry Points:**
- `python main_playlist.py` — single playlist download
- `python main_loop_playlist.py` — batch loop mode for multiple playlists
- `python main_loop_noplaylist.py` — batch loop for individual videos

**Steps:**
1. User runs playlist/batch variant
2. For single playlist: prompts for playlist URL
3. For loop mode: reads URLs from file or stdin
4. yt-dlp fetches playlist metadata (video count, titles, durations)
5. For each video in playlist/batch:
   - Fetch video details
   - User selects format (or uses default)
   - Download video
   - Apply polite delay between downloads
6. Progress reported per video
7. Summary of all downloaded files displayed

**Error Handling:**
- Individual video failure in playlist → log error, continue to next video
- Playlist not found → error message, skip
- Rate limiting → adaptive delays between downloads
- Keep yt-dlp updated to handle YouTube changes
