# Python-projects — 18 Beginner Scripts

Python 3.x scripts (calculator, games, face detection, QR, etc.).

## Stack
- `requests`, `opencv-python`, `matplotlib`, `pillow`, `qrcode`, `beautifulsoup4`, `PyDictionary`, `schedule`
- Lint: `ruff`; type check: `mypy`

## Commands
```bash
pip install -r requirements.txt
python basic_calculator.py
python qr_code_generator.py
ruff check .
mypy *.py
```

## Notes
- Standalone scripts; no formal tests
- PEP 8, `snake_case`
- README documents each script
