# Technology Stack — Python-projects

## Overview
Collection of 18 beginner-friendly Python scripts covering calculators, games, automation, and computer vision.

## Language & Runtime
| Technology | Purpose |
|------------|---------|
| Python 3.x | Runtime (Python 2 not supported) |

## Key Libraries
| Library | Version | Purpose |
|---------|---------|---------|
| requests | 2.32.3 | HTTP requests |
| opencv-python | 4.11.0 | Computer vision / face detection |
| matplotlib | 3.10.1 | Data visualization / plotting |
| pillow | 11.1.0 | Image processing |
| qrcode | 8.0 | QR code generation |
| beautifulsoup4 | 4.13.3 | Web scraping |
| numpy | 2.2.4 | Numerical computing |
| PyDictionary | 2.0.1 | Dictionary/thesaurus API |
| schedule | 1.2.2 | Task scheduling |

## Quality
| Tool | Purpose |
|------|---------|
| ruff | Linting |
| mypy | Type checking |

## Script Categories
- **Calculators**: Basic calculator, scientific calculator
- **Games**: Rock paper scissors, dice rolling, etc.
- **Automation**: Email sender, scheduled tasks
- **Vision**: Face detection, image processing
- **Web**: URL shortener, web scraper
- **Utility**: QR code generator, dictionary tool
