## Workflows

### Workflow 1: QR Code Generation

**Description:** User runs a script that generates a QR code image from input data.

**Entry Points:**
- `python qr_code_generator.py`

**Steps:**
1. User runs `python qr_code_generator.py`
2. Script prompts for data to encode (URL, text, etc.)
3. Script uses the `qrcode` library to generate QR code image
4. Script saves image to local filesystem
5. User is notified of output file path

**Sequence Diagram:**
```
User                    qr_code_generator.py        qrcode library          Filesystem
  |                            |                         |                     |
  |-- python script.py ------->|                         |                     |
  |                            |                         |                     |
  |<-- "Enter data to --------|                         |                     |
  |     encode:"              |                         |                     |
  |-- user input ------------->|                         |                     |
  |                            |                         |                     |
  |                            |-- qrcode.make(data) --->|                     |
  |                            |<-- QRCodeImage ---------|                     |
  |                            |                         |                     |
  |                            |-- save image ---------->|                     |
  |                            |   (qrcode.png)          |                     |
  |<-- "QR code saved to -----|                         |                     |
  |     output.png"           |                         |                     |
```

**Error Handling:**
- Invalid input → prompt again with validation message
- File write error → catch exception, report path issue
- Missing `qrcode` library → `pip install qrcode` prompt

**Test Patterns:**
- `ruff check .` — linting for all scripts
- `mypy *.py` — type checking
- Manual testing via script execution

### Workflow 2: Face Detection from Webcam/Image

**Description:** User runs face detection script that identifies faces in images or webcam feed.

**Entry Points:**
- `python python_face_detection.py`

**Steps:**
1. User runs the face detection script
2. Script asks for input source (image file or webcam)
3. For image: loads file using OpenCV
4. For webcam: captures live video feed
5. Script uses Haar cascades or deep learning model via `opencv-python`
6. Detected faces are highlighted with rectangles
7. Result is displayed in a window and optionally saved

**Error Handling:**
- Camera not available → fallback to image mode
- No faces detected → inform user, no crash
- Missing OpenCV → `pip install opencv-python` message
