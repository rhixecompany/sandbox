# ecom
**Python/Django**: conventions, PEP 8, DRF ViewSets + Serializers, SimpleJWT.
**React/Redux**: react-app ESLint; `constants/actions/reducers` → `components`; React Bootstrap; Axios with JWT.
**Structure**: `frontend/src/components/`, `frontend/src/screens/`, `frontend/src/{actions,reducers,constants}/`.
**Security**: no `SECRET_KEY`; CORS via django-cors-headers; DRF serializer validation.
