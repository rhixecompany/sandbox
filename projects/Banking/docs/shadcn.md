# shadcn/ui Usage (Banking)

This project uses **shadcn/ui** components from `components/ui`.

## Add a component

```bash
npx shadcn@latest add button
```

## Customize styles

- Edit the component in `components/ui`.
- Tailwind config and globals are in `app/globals.css`.

## Notes

- Keep components server‑first; add `"use client"` only when needed.
- Follow project conventions in `AGENTS.md` and `.github/copilot-instructions.md`.
