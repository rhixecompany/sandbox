# shadcn/studio Documentation

## Introduction

shadcn/studio is an open-source & premium collection of copy-and-paste shadcn components, blocks, and templates - paired with a powerful theme generator & AI tools to craft, customize, and ship faster.

## What is shadcn/studio?

This is NOT a traditional component library or a replacement for shadcn/ui. Instead, it's a unique collection offering customizable variants of components, blocks, and templates. Preview, customize, and copy-paste them into your apps with ease.

Building on the solid foundation of shadcn components & blocks, shadcn/studio enhances it with custom-designed components & blocks to give you a head start.

## How to Use

### Using CLI

shadcn/studio uses shadcn CLI, offering an easy way to install components.

1. Visit the Components page to explore component variants
2. Select the desired CLI version
3. Copy the CLI command by clicking the copy icon
4. Run the copied command in your terminal

### Copy the Code

1. Choose your desired component (e.g., Accordion)
2. Hover over the component variant and click the Code button
3. Copy the code and paste it into your project

### Using Copy Prompt

The Copy Prompt feature allows you to quickly copy AI-optimized prompts:

1. Hover over the component variant
2. Click the Copy Prompt button
3. Paste the copied prompt into your IDE
4. Let the AI editor do the rest

### Using v0

v0 integration allows seamless import into Vercel's v0 platform:

1. Hover over the component variant
2. Click the "v0" button
3. It will automatically open in v0 with pre-configured code

## Important Note

In recent shadcn releases, an update was introduced to the tooltip component to address an issue related to TooltipProvider. Projects using the Tooltip component without an explicitly defined TooltipProvider may encounter errors.

**Solution**: Ensure that TooltipProvider is added at the appropriate level in your layout file.
