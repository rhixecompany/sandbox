# MCP Servers — Agent Context

## Overview
Multi-language MCP (Model Context Protocol) server implementations. Each directory contains a complete, buildable MCP server in its respective language.

## Languages
- TypeScript, Python, Go, Rust, Java, Kotlin, PHP, Ruby, Swift, C#

## Development
- Each language directory is self-contained with its own build system
- Refer to each subdirectory's README for language-specific setup
