# User Guide

## Getting Started

### Creating an Account

1. Navigate to the sign-up page
2. Enter your university email
3. Create a password
4. Fill in your university ID and name
5. Upload your university card image
6. Wait for admin approval (PENDING status)

### Signing In

1. Go to the sign-in page
2. Enter your email and password
3. Redirected to dashboard on success

## Features

### Browsing the Library

- Visit the Library page to see all available books
- Use search to find books by title or author
- Click on a book to view details

### Borrowing Books

1. Find a book in the library
2. Click on the book to open details
3. Click "Borrow" to request borrowing
4. Wait for admin approval
5. Check your profile for borrow status

### Returning Books

1. Go to My Profile
2. Find books you're currently borrowing
3. Click "Return" to return the book

### User Profile

- View your account information
- See your borrowing history
- Check currently borrowed books
- Update your display name

## Roles and Permissions

### Regular User
- Browse library catalog
- Request book borrowing
- View own profile and history
- Cannot access admin panel

### Administrator
- All user permissions
- Approve/reject user registrations
- Manage book catalog (add, edit, delete)
- Process book borrowing requests
- View system analytics

## Troubleshooting

### Can't Sign In
- Verify your account is approved (check email)
- Ensure credentials are correct
- Contact admin if issues persist

### Can't Borrow Books
- Verify you have no overdue books
- Check book availability
- Ensure your account is active

### Book Not Available
- Join waitlist if available
- Check back later for returns
- Contact admin for assistance
