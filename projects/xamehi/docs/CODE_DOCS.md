# Xamehi — Code Documentation

## Overview

Xamehi is a Django-based manga and comic reading platform with a React frontend. The backend serves comic content via Django REST Framework, while the frontend provides a modern reading interface. Currently at initial project scaffold stage.

**Stack:** Django 4.x | Python 3.10+ | React 18 | PostgreSQL | Django REST Framework  
**Status:** Initial Scaffold — Active Development

## 2. Planned Model Architecture

### Comic Models (to be implemented)

```python
from django.db import models
from django.contrib.auth.models import User
from django.core.validators import MinValueValidator, MaxValueValidator

class Comic(models.Model):
    """Manga/comic series model.

    Represents a comic series with metadata, cover art, and categorization.

    Attributes:
        title: Official series title.
        alt_title: Alternative title (subtitle, Japanese title).
        slug: URL-friendly identifier, auto-generated from title.
        description: Series synopsis/description.
        cover_image: Series cover/thumbnail image.
        status: Publication status (Ongoing, Completed, Hiatus, Dropped).
        author: Series author/creator name.
        artist: Series artist name (may differ from author).
        type: Comic type (Manga, Manhwa, Manhua, Western Comic).
        rating: Average user rating (0.0 - 10.0).
        view_count: Total page view counter.
        created_at: Auto-set creation timestamp.
        updated_at: Auto-set update timestamp.
    """
    STATUS_CHOICES = [
        ('ongoing', 'Ongoing'),
        ('completed', 'Completed'),
        ('hiatus', 'Hiatus'),
        ('dropped', 'Dropped'),
    ]

    TYPE_CHOICES = [
        ('manga', 'Manga'),
        ('manhwa', 'Manhwa'),
        ('manhua', 'Manhua'),
        ('comic', 'Western Comic'),
    ]

    title = models.CharField(max_length=255, unique=True)
    alt_title = models.CharField(max_length=500, blank=True)
    slug = models.SlugField(max_length=255, unique=True)
    description = models.TextField(blank=True)
    cover_image = models.URLField(max_length=500, blank=True)
    status = models.CharField(max_length=20, choices=STATUS_CHOICES, default='ongoing')
    author = models.CharField(max_length=255, blank=True)
    artist = models.CharField(max_length=255, blank=True)
    type = models.CharField(max_length=20, choices=TYPE_CHOICES, default='manga')
    rating = models.DecimalField(max_digits=3, decimal_places=1, default=0.0)
    view_count = models.IntegerField(default=0)

    created_at = models.DateTimeField(auto_now_add=True)
    updated_at = models.DateTimeField(auto_now=True)

    class Meta:
        ordering = ['-created_at']
        indexes = [
            models.Index(fields=['slug']),
            models.Index(fields=['status']),
            models.Index(fields=['title']),
        ]

    def __str__(self) -> str:
        return self.title


class Genre(models.Model):
    """Comic genre/category taxonomy.

    Genres are used for filtering and discovery.
    """
    name = models.CharField(max_length=50, unique=True)
    slug = models.SlugField(max_length=50, unique=True)
    description = models.TextField(blank=True)

    class Meta:
        ordering = ['name']

    def __str__(self) -> str:
        return self.name


class ComicGenre(models.Model):
    """Many-to-many relationship between comics and genres."""
    comic = models.ForeignKey(Comic, on_delete=models.CASCADE)
    genre = models.ForeignKey(Genre, on_delete=models.CASCADE)

    class Meta:
        unique_together = ('comic', 'genre')

    def __str__(self) -> str:
        return f"{self.comic.title} - {self.genre.name}"


class Chapter(models.Model):
    """Individual chapter within a comic series.

    Each chapter has sequential numbering, multiple pages,
    and tracks view statistics.

    Attributes:
        comic: FK to parent Comic series.
        title: Chapter title (e.g., "Volume 1: The Beginning").
        chapter_number: Sequential chapter number.
        volume_number: Optional volume grouping.
        page_count: Number of pages in the chapter.
        release_date: Official release/publication date.
        views: View counter for this chapter.
        created_at: Auto-set creation timestamp.
    """
    comic = models.ForeignKey(Comic, on_delete=models.CASCADE, related_name='chapters')
    title = models.CharField(max_length=255)
    chapter_number = models.DecimalField(max_digits=6, decimal_places=1)
    volume_number = models.IntegerField(null=True, blank=True)
    page_count = models.IntegerField(default=0)
    release_date = models.DateField(null=True, blank=True)
    views = models.IntegerField(default=0)

    created_at = models.DateTimeField(auto_now_add=True)
    updated_at = models.DateTimeField(auto_now=True)

    class Meta:
        ordering = ['comic', 'chapter_number']
        unique_together = ('comic', 'chapter_number')
        indexes = [
            models.Index(fields=['comic', 'chapter_number']),
        ]

    def __str__(self) -> str:
        return f"{self.comic.title} Ch.{self.chapter_number}"


class Page(models.Model):
    """Individual page/image within a chapter.

    Images are stored as URLs (external hosting or local media).
    Ordered by page_number for sequential reading.

    Attributes:
        chapter: FK to parent Chapter.
        page_number: Sequential page number within the chapter.
        image_url: URL to the page image (external or local).
        width: Optional image width in pixels.
        height: Optional image height in pixels.
    """
    chapter = models.ForeignKey(Chapter, on_delete=models.CASCADE, related_name='pages')
    page_number = models.IntegerField()
    image_url = models.URLField(max_length=500)
    width = models.IntegerField(null=True, blank=True)
    height = models.IntegerField(null=True, blank=True)

    class Meta:
        ordering = ['page_number']
        unique_together = ('chapter', 'page_number')

    def __str__(self) -> str:
        return f"{self.chapter} - Page {self.page_number}"
```

### User Interaction Models (planned)

```python
class Bookmark(models.Model):
    """User bookmark tracking reading progress per comic."""
    STATUS_CHOICES = [
        ('reading', 'Reading'),
        ('completed', 'Completed'),
        ('plan_to_read', 'Plan to Read'),
        ('dropped', 'Dropped'),
    ]

    user = models.ForeignKey(User, on_delete=models.CASCADE)
    comic = models.ForeignKey(Comic, on_delete=models.CASCADE)
    status = models.CharField(max_length=20, choices=STATUS_CHOICES, default='reading')
    current_chapter = models.ForeignKey(Chapter, on_delete=models.SET_NULL, null=True, blank=True)
    current_page = models.IntegerField(default=1)
    notes = models.TextField(blank=True)

    created_at = models.DateTimeField(auto_now_add=True)
    updated_at = models.DateTimeField(auto_now=True)

    class Meta:
        unique_together = ('user', 'comic')

    def __str__(self) -> str:
        return f"{self.user.username} - {self.comic.title}"


class Comment(models.Model):
    """User comment on a chapter."""
    user = models.ForeignKey(User, on_delete=models.CASCADE)
    chapter = models.ForeignKey(Chapter, on_delete=models.CASCADE, related_name='comments')
    content = models.TextField()
    parent = models.ForeignKey('self', on_delete=models.CASCADE, null=True, blank=True, related_name='replies')

    is_edited = models.BooleanField(default=False)
    created_at = models.DateTimeField(auto_now_add=True)
    updated_at = models.DateTimeField(auto_now=True)

    class Meta:
        ordering = ['-created_at']

    def __str__(self) -> str:
        return f"{self.user.username} on {self.chapter}"


class Rating(models.Model):
    """User rating for a comic."""
    user = models.ForeignKey(User, on_delete=models.CASCADE)
    comic = models.ForeignKey(Comic, on_delete=models.CASCADE, related_name='ratings')
    score = models.IntegerField(validators=[MinValueValidator(1), MaxValueValidator(10)])
    review = models.TextField(blank=True)

    created_at = models.DateTimeField(auto_now_add=True)

    class Meta:
        unique_together = ('user', 'comic')

    def __str__(self) -> str:
        return f"{self.user.username} - {self.comic.title}: {self.score}/10"
```

## 3. Planned API Endpoints

```
Authentication:
  POST /api/auth/register/     → User registration
  POST /api/auth/login/        → JWT login
  POST /api/auth/refresh/      → Token refresh

Comics:
  GET    /api/comics/          → Paginated comic list (filter by genre, status, type)
  GET    /api/comics/{slug}/   → Comic detail with chapter list
  POST   /api/comics/          → Create comic (admin)
  PUT    /api/comics/{slug}/   → Update comic (admin)
  DELETE /api/comics/{slug}/   → Delete comic (admin)

Chapters:
  GET    /api/comics/{slug}/chapters/         → Chapter list for comic
  GET    /api/comics/{slug}/chapters/{num}/   → Chapter detail with pages
  POST   /api/comics/{slug}/chapters/         → Create chapter (admin)

User:
  GET    /api/bookmarks/       → User's bookmarks
  POST   /api/bookmarks/       → Add bookmark
  DELETE /api/bookmarks/{id}/  → Remove bookmark
  GET    /api/comments/        → User's comments
  POST   /api/comments/        → Post comment
  GET    /api/ratings/         → User's ratings
  POST   /api/ratings/         → Rate a comic
```
