# User Guide — Xamehi

Welcome to Xamehi! This guide covers how to use the application from an end-user perspective.

## Overview

Xamehi is a web application that provides real-time currency exchange rates and crypto news. It features a modern React frontend with an Express.js backend API.

## Features

- **Currency Exchange Rates**: Get real-time currency conversion rates using the Alpha Vantage API
- **Crypto News**: Stay updated with the latest cryptocurrency news
- **Responsive Design**: Works on desktop and mobile browsers

## Getting Started

### Accessing the Application

1. Open your web browser
2. Navigate to the Xamehi application URL
3. The React frontend will load automatically

### Using Currency Exchange

1. Navigate to the currency exchange section
2. Select the source currency (from currency)
3. Select the target currency (to currency)
4. View the real-time exchange rate

### Viewing Crypto News

1. Navigate to the news section
2. Browse the latest cryptocurrency news articles
3. News is fetched from live crypto news sources

## Troubleshooting

- **Page not loading**: Check your internet connection and refresh the page
- **Exchange rate not showing**: The API may be temporarily unavailable; try again in a few moments
- **News not loading**: Check your network connection and try refreshing
