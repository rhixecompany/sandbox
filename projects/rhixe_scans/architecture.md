# Rhixe Scans — Architecture Overview

## Overview
Full-stack scan processing and document management application with a Next.js frontend and Python backend.

## Key Components
- **backend/** — Python backend (API, processing, database)
- **frontend/** — Next.js frontend with React components
- **bash/** — Shell scripts for deployment and maintenance
- **compose/** — Docker Compose configurations for local/dev/production
- **bin/** — Binary/executable scripts

## Technology
- Frontend: Next.js (React, Radix UI, Prisma, PayPal)
- Backend: Python (Django or FastAPI)
- Infrastructure: Docker Compose, systemd
- Database: PostgreSQL (via Prisma)
