# User Guide

Welcome to Rhixescans! This guide will help you get started with our comic reading platform.

## Getting Started

### Creating an Account

1. Click **Sign Up** in the navigation bar
2. Enter your email address
3. Create a password
4. Click **Create Account**

Alternatively, you can sign in with your GitHub account.

### Signing In

1. Click **Sign In**
2. Enter your email and password
3. Click **Sign In** button

## Browsing Comics

### Home Page

The home page displays a curated list of comics. You can:
- Browse featured comics
- Filter by genre
- Search for specific titles

### Search

Use the search bar to find comics by:
- Title
- Author
- Artist
- Genre

### Filtering

Filter comics by:
- **Genre**: Action, Romance, Fantasy, etc.
- **Status**: Ongoing, Completed, Hiatus
- **Category**: Webtoon, Manga, Manhwa

## Reading Comics

### Selecting a Comic

1. Click on any comic cover
2. View the comic details page
3. Click **Start Reading** or select a chapter

### Chapter Navigation

- Use the chapter list to jump to specific chapters
- Previous/Next buttons for sequential reading
- Progress is automatically saved

### Image Viewer

- Click on images to view fullscreen
- Use arrow keys or on-screen buttons to navigate
- Zoom in/out with scroll wheel or pinch

## Bookmarks

### Saving Comics

Click the **Bookmark** icon on any comic to save it for later.

### Managing Bookmarks

1. Click your profile icon
2. Select **Bookmarks**
3. View all saved comics
4. Click **Remove** to delete a bookmark

## Profile Management

### Updating Profile

1. Click your profile icon
2. Select **Settings**
3. Update your name, email, or profile picture
4. Click **Save Changes**

### Changing Password

1. Go to **Settings**
2. Click **Change Password**
3. Enter current password
4. Enter new password and confirm

## Admin Features

If you have admin access, you can:

### Managing Comics

- Add new comics
- Edit comic details
- Delete comics
- Upload chapter images

### Managing Chapters

- Add chapters to comics
- Edit chapter details
- Reorder chapters
- Delete chapters

### Dashboard

Access the admin dashboard at `/admin` to manage all content.

## Troubleshooting

### Can't Sign In

- Verify your email and password
- Try resetting your password
- Check if your account is active

### Images Not Loading

- Clear your browser cache
- Check your internet connection
- Try a different browser

### Slow Performance

- Close unnecessary tabs
- Use a modern browser
- Check your internet speed
