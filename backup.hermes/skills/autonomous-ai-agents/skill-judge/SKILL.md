# Skill Judge

## Description

Evaluate and score skills or outputs against defined quality criteria. Provides structured evaluation and scoring for quality assurance.

## When to Use

- Reviewing skill design quality
- Auditing existing skills
- Improving SKILL.md files
- Evaluating skill packages
- Ensuring skill standards compliance
- Planning skill improvements

## When NOT to Use

- Using skills (not evaluating)
- Non-skill content
- Quick feedback without rigor
- Real-time skill development

## Workflow

### Phase 1: Identify Skill

- Select skill to evaluate
- Gather skill documentation
- Review skill structure
- Note current state

### Phase 2: Evaluate Quality

- Check against specifications
- Assess best practices
- Score multiple dimensions
- Identify issues

### Phase 3: Provide Feedback

- Document findings
- Suggest improvements
- Prioritize changes
- Provide actionable guidance

### Phase 4: Plan Improvements

- Create improvement plan
- Prioritize changes
- Estimate effort
- Track progress

## Best Practices

- Use consistent evaluation criteria
- Provide specific feedback
- Suggest concrete improvements
- Consider skill purpose
- Evaluate against standards
- Document evaluation results
- Track improvements over time

## Overview

Skill evaluation and scoring tool for quality assurance. Provides structured multi-dimensional assessment against standards, with actionable feedback and improvement tracking.

## Verification Checklist

- [ ] Evaluation criteria are consistent and applied uniformly
- [ ] Feedback is specific, actionable, and prioritized
- [ ] Skill purpose and context are considered in evaluation
- [ ] Improvement plan has effort estimates and tracking
- [ ] Evaluation results are documented for future reference
