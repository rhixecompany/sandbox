## Description

Create, read, edit, and manipulate Word documents (.docx files). Includes formatting, tables of contents, headings, page numbers, letterheads, tracked changes, and comments.

## When to Use

- Creating or editing Word documents (.docx)
- Generating professional reports or memos
- Extracting text from existing .docx files
- Combining or splitting Word documents
- Adding formatting, headers, footers
- Working with tracked changes or comments
- Creating templates or letterheads

## When NOT to Use

- PDF files (use pdf skill)
- Spreadsheets (use xlsx skill)
- Google Docs or other cloud editors
- Plain text files
- Code documentation

## Workflow

### Phase 1: Prepare Document

- Determine document type and purpose
- Gather content and sources
- Plan structure and layout

### Phase 2: Create/Edit

- Create new document or open existing
- Add content and formatting
- Insert tables, images, headers/footers
- Apply styles and themes

### Phase 3: Format & Polish

- Adjust fonts and spacing
- Create table of contents
- Add page numbers
- Apply consistent formatting

### Phase 4: Finalize

- Review for completeness
- Check formatting consistency
- Add tracked changes if collaborative
- Export or save

## Tools & References

- **Related Skills**: pdf, project-docs
- **File Format**: .docx (Office Open XML)
- **Libraries**: python-docx, docxtpl
- **Templates**: Built-in and custom

## Best Practices

- Use styles for consistent formatting
- Create table of contents for long docs
- Use headers/footers for professional look
- Keep file sizes reasonable
- Use tracked changes for collaboration
- Save frequently
- Back up important documents
