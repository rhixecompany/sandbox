## Description

Knowledge and utilities for creating animated GIFs optimized for Slack. Provides constraints, validation tools, and animation concepts.

## When to Use

- Creating animated GIFs for Slack
- Making GIFs of specific actions or concepts
- Optimizing GIFs for Slack display
- Creating reaction GIFs
- Making demo or tutorial GIFs

## When NOT to Use

- Static images (use image tools)
- Video files (use video tools)
- Non-Slack GIFs
- Real-time animation

## Workflow

### Phase 1: Plan Animation

- Identify animation concept
- Plan frames and timing
- Determine dimensions
- Check Slack constraints

### Phase 2: Create Animation

- Generate frames
- Set timing and delays
- Optimize file size
- Validate format

### Phase 3: Optimize for Slack

- Resize to Slack dimensions
- Compress file size
- Test in Slack
- Verify playback

### Phase 4: Deploy & Share

- Upload to Slack
- Test in channels
- Get feedback
- Archive GIF

## Tools & References

- **Related Skills**: frontend-design, canvas-design
- **Slack Constraints**: Size, format, duration limits
- **Tools**: GIF creation and optimization tools
- **Formats**: GIF, WebP, MP4

## Best Practices

- Keep animations short (2-5 seconds)
- Optimize file size for Slack
- Use clear, readable content
- Test in Slack before sharing
- Document animation concept
- Archive GIFs for reuse
- Get feedback from team
