## Description

Suite of tools for creating elaborate, multi-component HTML artifacts using modern frontend web technologies (React, Tailwind CSS, shadcn/ui). Use for complex artifacts requiring state management, routing, or shadcn/ui components.

## When to Use

- Creating complex HTML artifacts
- Building multi-component interfaces
- Using React with state management
- Implementing routing in artifacts
- Using shadcn/ui components
- Creating interactive dashboards
- Building web applications

## When NOT to Use

- Simple single-file HTML/JSX
- Static content
- Non-interactive pages
- Real-time collaborative editing

## Workflow

### Phase 1: Plan Artifact

- Define artifact purpose
- Identify components needed
- Plan state management
- Determine routing needs

### Phase 2: Design Architecture

- Choose component structure
- Plan state flow
- Design routing
- Identify dependencies

### Phase 3: Implement

- Create React components
- Implement state management
- Add routing if needed
- Style with Tailwind/shadcn

### Phase 4: Test & Deploy

- Test all functionality
- Verify responsive design
- Test in target environment
- Deploy artifact
