## Description

Creating algorithmic art using p5.js with seeded randomness and interactive parameter exploration. Generate original algorithmic art, flow fields, and particle systems.

## When to Use

Use this skill when:

- Creating art using code
- Generating algorithmic art
- Building flow fields
- Creating particle systems
- Exploring generative design
- Building interactive visualizations

**Triggers**: "Create algorithmic art", "Build a generative art piece", "Make a flow field", "Create a particle system", "Generate art with code"

## When NOT to Use

- For copying existing artists' work (avoid copyright violations)
- For static design (use canvas-design instead)
- For web UI (use frontend-design instead)
- For data visualization (use mermaid-diagrams instead)

## Workflow

### Phase 1: Concept

- Define artistic intent
- Choose algorithm type
- Plan parameter space
- Sketch initial design

### Phase 2: Implementation

- Set up p5.js canvas
- Implement core algorithm
- Add randomness with seeding
- Create parameter controls

### Phase 3: Refinement

- Adjust colors and shapes
- Optimize performance
- Add interactivity
- Test parameter ranges

### Phase 4: Export

- Generate final artwork
- Export as PNG or SVG
- Document parameters
- Share results
