## Description

Create beautiful visual art in .png and .pdf documents using design philosophy. Create original visual designs, never copying existing artists' work to avoid copyright violations.

## When to Use

Use this skill when:

- Creating posters
- Designing visual art
- Building infographics
- Creating visual designs
- Designing printed materials
- Creating branded graphics
- Building visual presentations

**Triggers**: "Create a poster", "Design this visual", "Make an infographic", "Create visual art", "Design a graphic"

## When NOT to Use

- For copying existing artists' work (copyright violation)
- For web UI (use frontend-design instead)
- For algorithmic art (use algorithmic-art instead)
- For presentations (use marp-slide instead)

## Workflow

### Phase 1: Concept

- Understand purpose
- Define audience
- Sketch ideas
- Plan layout

### Phase 2: Design

- Choose color scheme
- Select typography
- Create layout
- Add visual elements

### Phase 3: Refinement

- Adjust proportions
- Refine colors
- Improve typography
- Add details

### Phase 4: Export

- Export as PNG/PDF
- Verify quality
- Test on different devices
- Archive design files

- **Original Work**: Create original designs, don't copy
- **Color Theory**: Use complementary colors
- **Typography**: Choose readable fonts
- **Hierarchy**: Guide viewer's eye
- **Simplicity**: Less is more
- **Testing**: Verify on different devices
