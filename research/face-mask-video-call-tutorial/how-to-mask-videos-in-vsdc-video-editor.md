## Overview

Video masking outlines an area to modify independently. VSDC provides clipping masks, inverted masks, and text masks.

> Note: Masking requires VSDC Pro ($19.99/year). Free alternative available for face blurring.

## Inverted Mask (Pixelization) Steps

1. Add video → Video Effects → Filters → Pixelize
2. Select masking area: Ellipse, Rectangle, or Free shape
3. Double-click Pixelize layer → Add Ellipse object
4. Right-click Ellipse → Properties → Composition mode → Mask → Inverted mask → Yes

## Motion Tracking

To make mask follow face movements:

1. Right-click video → "Create a movement map"
2. Place tracking frame over face
3. Press Start analysis
4. Apply editing → Yes
5. Double-click video → Pixelize → Ellipse → Add object → Movement → Movement map
6. Preview to verify

## Applications

- Face blurring for privacy
- Color changes on specific areas
- Object cutting/replacement
- Text masks
